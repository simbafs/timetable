# 如何新增學校支援

這個目錄存放所有學校的課表設定檔。若要新增一所學校，請依照以下步驟操作：

## 步驟

1.  在 `src/data/` 目錄下建立一個新的 `.yaml` 檔案。
    - **檔名規範**：請使用學校的英文縮寫（小寫），例如 `ntu.yaml`。這個檔名將會成為系統中的學校 ID。

2.  依照下方的格式填寫內容。

## 檔案格式

```yaml
name: 學校名稱 (例如：國立臺灣大學)
time:
    # 節次代碼: 時間範圍 (HHMM-HHMM)
    1: 0810-0900
    2: 0910-1000
    N: 1220-1310
    # ... 其他節次
semester:
    # 學期代碼: 學期區間 (YYYY/MM/DD-YYYY/MM/DD)
    # 系統會自動選取字典序最大的學期作為預設學期
    113-2: 2025/02/17-2025/06/20
```

## 欄位說明

- **name**: 學校的中文全名，將顯示於選單中。
- **time**: 定義每一節次的時間。
    - `key`: 節次代碼（如 1, 2, A, B, N 等），顯示在課表左側。
    - `value`: 時間範圍，格式必須為 `HHMM-HHMM`（24小時制）。
- **semester**: 定義學期開始與結束日期。
    - `key`: 學期代碼（如 113-2）。
    - `value`: 日期範圍，格式必須為 `YYYY/MM/DD-YYYY/MM/DD`。
        - **開始日期**：請填寫第一週的週一（即便當天放假）。
        - **結束日期**：請填寫最後一週的週五。
    - **注意**：系統會自動抓取最新的學期（即列表中最後一個項目），因此新增學期時請直接加在列表下方。

## 範例

以 `ncku.yaml` (成功大學) 為例：

```yaml
name: 國立成功大學
time:
    0: 0700-0750
    1: 0800-0850
    2: 0900-0950
    # ... 中略
    N: 1210-1300
    5: 1310-1400
    # ... 中略
semester:
    114-2: 2026/02/23-2026/06/26
```
